# Manipulando Arquivos Via Linha de Comandos

***Conteúdo da Aula:***

Comandos usados:

* cp .\Cmder.zip F:\minha-pasta\;
* ls E:\minha-pasta\;
* rm E:\minha-pasta\Cmder.zip;
* mv .\Cmder.zip E:\minha-pasta\terminal.zip;
* mv E:\minha-pasta\terminal.zip Cmder.zip;
